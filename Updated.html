Background

ABC Technologies suspects that a recently resigned employee, working on the highly confidential "Project Falcon", accessed, staged, and possibly exfiltrated proprietary intellectual property before leaving the company.

During a preliminary review conducted by the IT department, investigators identified a project workspace located under C:\Projects containing multiple department folders, including Falcon, Finance, HR, and Archive. Management believes that files associated with the Falcon project may contain confidential business information relevant to the investigation.

Scenario Details

Management observed unusual off-hours file access, cleared browser history, and unauthorized USB mass storage device usage on the employee's workstation (WIN-FALCON01).

Objectives & Rules of Engagement

  • Verify the integrity of provided forensic images.
  • Identify the primary user and suspicious files.
  • Analyze Registry and Event Logs for USB insertion activity.
  • Analyze browser artifacts for cloud exfiltration evidence.
  • Build a chronological timeline of the incident.
  • Draft a professional Final Report.

How to Build the Timeline

The timeline should reconstruct the user's activities in chronological order using forensic evidence.

For each event:

  • Identify the timestamp.
  • Describe the activity.
  • Cite the supporting artifact.
Example:
[2026-06-10 14:06:01 IST] User searched for a cloud storage service. Source: Browser History
[2026-06-10 14:10:44 IST] PuTTY downloaded. Source: Chrome Downloads
[2026-06-10 14:53:00 IST] USB device connected. Source: USBSTOR
